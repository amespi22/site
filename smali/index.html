Learning Smali

  • If you need to take a .java file to a .smali file:

    • Compile for java 7: javac test.java -source 1.7 -target 1.7
    • Run dx to get a dex file: dx --dex --output=classes.dex test.class. The dx.jar file may be around /opt/android-sdk/build-tools/
    • Run baksmali classes.dex. You may want to give it the -o flag so you can tell it where to output the java file.
